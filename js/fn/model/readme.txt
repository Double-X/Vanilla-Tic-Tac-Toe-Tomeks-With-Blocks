FMIsDraw is responsible for checking whether the game's a draw.
FMIsSpecialSq is responsible for checking whether a given square is special, like being a tomek or block.
FMPlayer is responsible for returning the player given the move number.
FMRenderCmds is responsible for constructing commands that are issued to the views.
FMSqs is responsible for returning the current and last 1D square state list.
FMWinner is responsible for returning the winner with the winning squares.
FMWinningLs is responsible for returning lists of squares that can be winning lines.
